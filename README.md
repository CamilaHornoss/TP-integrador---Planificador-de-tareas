Este sistema de gestión de tareas facilita el orden y la organización diaria según prioridad.
Permite agregar, visualizar, priorizar, completar, eliminar, buscar y deshacer/rehacer acciones sobre tareas, utilizando estructuras dinámicas (listas, pilas y colas), ademas de estructuraas de Corte de control, recursividad, TAD, algoritmos de ordenamiento y y archivos para persistir la información entre ejecuciones.

Archivos necesarios:
Descomprimir el archivo .zip y colocar todos los archivos del proyecto dentro de una misma carpeta:
Todos los ".c"
Todos los ".h"
(Opcional) "tareas.dat"
Si no existe (lo esperado para el primer uso), en el programa inicia "vacío", y da la opción de crear uno al ir agregando tareas, para guardar las mismas.

Modo de uso:
El programa tiene un menú intuitivo, por lo que su uso es sencillo.
